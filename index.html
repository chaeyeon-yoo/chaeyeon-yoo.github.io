02 — Skills

기술 스택

실무에서 직접 검증한 기술들입니다.

HTML5 / 마크업
웹 표준을 준수한 시맨틱 마크업을 기반으로 구조 설계 및 유지보수가 용이한 코드 작성을 지향합니다.
CSS3 / 레이아웃
Flex와 Grid를 활용한 반응형 레이아웃 구현이 가능하며, 일관된 스타일 가이드 기반의 UI 구현 경험을 보유하고 있습니다.
웹 접근성 (WCAG)
웹 접근성 지침(WCAG)을 고려한 마크업 및 UI 개선 경험이 있으며, 스크린리더 대응 및 키보드 접근성 구현이 가능합니다.
반응형 / 크로스브라우징
다양한 디바이스 환경을 고려한 반응형 웹 구현이 가능하며, 주요 브라우저 간 호환성을 고려한 퍼블리싱을 진행합니다.
JavaScript
기본적인 DOM 제어 및 이벤트 처리가 가능하며, 기존 코드의 유지보수 및 간단한 인터랙션 구현 경험이 있습니다. jQuery를 활용한 UI 인터랙션 구현 및 레거시 코드 유지보수 경험도 보유하고 있습니다.
Git
Git을 활용한 버전 관리 및 협업 경험이 있습니다. Figma 시안을 바탕으로 디자인 의도를 파악하고 마크업으로 구현하는 협업 과정을 경험했습니다.
